# PWA Icons

Place the following icon files here before deploying:

- `icon-192.png` — 192×192 px (required for PWA)
- `icon-512.png` — 512×512 px (required for PWA splash)
- `apple-icon.png` — 180×180 px (Apple Touch Icon)

Recommended tool: https://realfavicongenerator.net
Use the VegSage brand green (#22c55e) as background color.
